# 任务：基于 Vue3 开发一款功能完整、界面简约高级、功能完整的 TODO List 单页应用

## 一、项目概述
这是一个展示“一周内掌握 Vue3 核心语法 + 组合式 API”学习成果的个人项目，需体现技术深度与产品完成度。  
要求：**功能实用、交互流畅、视觉简约、代码结构清晰**，适合放入简历项目展示。

## 二、技术栈
- Vue 3（组合式 API + `<script setup>` 语法）
- 状态管理：Pinia（或 Vuex 4）
- UI 组件：自定义 + 可选用轻量组件库（如 Element Plus 或 Naive UI 的最小化使用）
- 图标：Lucide 或 Iconify
- 样式：Tailwind CSS（或纯 CSS 变量 + Flex/Grid 布局）
- 本地持久化：localStorage
- 日期处理：dayjs
- 构建工具：Vite

## 三、核心功能需求
### 1. 任务管理（增删改查）
- 添加任务：输入框 + 回车/点击添加，支持优先级（高/中/低）或标签
- 列表展示：任务标题、创建时间、优先级标签、完成状态
- 编辑任务：点击任务文本进入编辑模式（行内编辑）
- 删除任务：单个删除 + 批量删除勾选的任务
- 标记完成：复选框切换完成状态，视觉区分（如颜色、删除线）

### 2. 提醒功能
- 可为任务设置提醒时间（日期+时间选择器）
- 临近提醒（如提前 10 分钟）在页面要有醒目的 Toast 提示
- 过期任务自动标记为“已过期”样式

### 3. 收藏/置顶功能
- 每个任务可点击“星标”收藏，收藏任务置顶显示
- 收藏状态持久化

### 4. 状态管理与筛选
- 支持查看：全部/未完成/已完成/已收藏/已过期，支持用户方便快捷的管理查看自己的list
- 支持按优先级、创建时间排序
- 搜索框：实时过滤任务标题

### 5. 数据持久化
- 使用 localStorage 保存所有任务数据
- 页面刷新后数据不丢失

## 四、UI/UX 设计要求
### 视觉风格
- 页面简洁而不失高级感，字体方案小众而高级，页面中展现的元素要有轻微的卡片阴影，可以考虑给适当的元素添加hover效果

### 布局结构
- 顶部：Logo（TODO Pro） + 搜索框 + 添加任务输入区
- 左侧：筛选选项卡（全部/进行中/已完成/收藏）
- 中间：任务列表（卡片式，每项包含：复选框、任务内容、优先级标签、操作按钮）
- 操作按钮组（每个任务）：编辑、删除、星标收藏、设置提醒等

### 交互细节
- 悬停效果：任务卡片轻微背景色变化
- 加载状态：数据加载时骨架屏
- 空状态：无任务时展示插画 + 提示文案
- 动画：任务添加/删除使用淡入淡出动画，切换状态有过渡效果
